# STUDENT-AUTHENTICATION
The Student Management System is a menu-driven command line project developed using Python. This application allows users to manage student records efficiently by performing various operations such as adding new students, viewing records, updating marks, deleting students, and generating results. The system collects student details including roll number, name, and marks for three subjects, then calculates total marks automatically. It provides useful academic insights such as class average, topper details, pass or fail status, grade calculation (A, B, C, D), total student count, and sorting students based on total marks. The project also includes a beep sound feature that works using the built-in winsound library on Windows systems, enhancing user interaction through audio feedback.

This project is built entirely using Python 3 and runs in a Command Line Interface (CLI) environment without requiring any external libraries. It demonstrates fundamental programming concepts such as functions, loops, conditional statements, exception handling, lists, dictionaries, sorting, and lambda functions. The grading system assigns Grade A for totals greater than or equal to 270, Grade B for totals greater than or equal to 240, Grade C for totals greater than or equal to 200, and Grade D for totals below 200. Students scoring 120 or more total marks are considered Pass, while others are marked as Fail.

To run the project, clone the repository from GitHub, navigate to the project folder, and execute the Python file using the command python studentauthetication.py. The system stores data temporarily during runtime, meaning all records are cleared once the program exits. Although it currently does not use file handling or database storage, future improvements may include adding persistent data storage, integrating a database such as MySQL or SQLite, developing a graphical user interface using Tkinter, implementing login authentication, and enabling result export to CSV format.

This project is suitable for beginners and students learning Python fundamentals, as it demonstrates real-world implementation of structured programming concepts in an organized and practical manner.
